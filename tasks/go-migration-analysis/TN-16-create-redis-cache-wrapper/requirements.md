# TN-16: Redis Cache Wrapper

## 1. Обоснование
Централизованная работа с кэшем через единый интерфейс.

## 2. Сценарий
Приложение кэширует результаты классификации в Redis.

## 3. Требования
- Обёртка над go-redis/v9.
- Методы Get/Set/Delete с TTL.
- Сериализация JSON.
- Graceful деградация при недоступности.
