# TN-15: Миграции в CI/CD

## 1. Обоснование
CI должен проверять, что миграции применяются корректно.

## 2. Сценарий
При запуске тестов в CI автоматически создаётся тестовая БД с миграциями.

## 3. Требования
- Тестовая БД в CI (PostgreSQL service).
- Применение миграций перед тестами.
- Проверка rollback миграций.

## 4. Критерии приёмки
- [x] CI создаёт тестовую БД. ✅ **УЖЕ БЫЛ НАСТРОЕН**
- [x] Миграции применяются автоматически. ✅ **ВЫПОЛНЕНО**
- [x] Тесты проходят на мигрированной БД. ✅ **ВЫПОЛНЕНО** (тесты запускаются после миграций)
- [x] Rollback тестируется. ✅ **ВЫПОЛНЕНО** (добавлен тест TestMigrationsRunDown)
