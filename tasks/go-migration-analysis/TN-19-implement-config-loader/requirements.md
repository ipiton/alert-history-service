# TN-19: Config Loader с Viper

## 1. Обоснование
Централизованное управление конфигурацией с поддержкой 12-factor.

## 2. Сценарий
Приложение загружает настройки из файла, ENV переменных с приоритетом ENV.

## 3. Требования
- Использовать Viper.
- Поддержка config.yaml и ENV.
- Валидация обязательных параметров.
- Типизированная конфигурация.

## 4. Критерии приёмки
- [ ] Config структура определена.
- [ ] LoadConfig функция реализована.
- [ ] ENV переменные перезаписывают файл.
- [ ] Валидация работает.
