# TN-30: Test Coverage Setup

## 1. Обоснование
Метрики покрытия кода тестами для контроля качества.

## 2. Сценарий
CI генерирует отчёт покрытия и показывает badge в README.

## 3. Требования
- Coverage отчёты в CI.
- Badge в README.
- Минимальный порог 80%.
- HTML отчёты для анализа.

## 4. Критерии приёмки
- [ ] Coverage собирается в CI.
- [ ] Badge отображается.
- [ ] HTML отчёт генерируется.
- [ ] Порог coverage проверяется.
