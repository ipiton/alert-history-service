# TN-27: Go Contributing Guide

## 1. Обоснование
Документация для новых разработчиков по работе с Go кодом.

## 2. Сценарий
Новый разработчик читает CONTRIBUTING.md и начинает контрибьютить.

## 3. Требования
- Стандарты кодирования.
- Процесс разработки.
- Тестирование.
- Code review checklist.

## 4. Критерии приёмки
- [ ] CONTRIBUTING-GO.md создан.
- [ ] Стандарты описаны.
- [ ] Примеры кода включены.
- [ ] Checklist для PR.
