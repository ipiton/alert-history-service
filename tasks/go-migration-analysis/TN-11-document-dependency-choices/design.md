# TN-11: Формат ADR

## Шаблон ADR
```markdown
# ADR-001: Выбор веб-фреймворка

## Статус
Принято

## Контекст
Необходим высокопроизводительный HTTP фреймворк.

## Решение
Выбран Fiber на основе fasthttp.

## Альтернативы
- Gin: популярный, но медленнее
- Echo: хороший, но меньше middleware

## Последствия
+ Высокая производительность
- Меньше примеров в сообществе
```
