# TN-13: SQLite Adapter для разработки

## 1. Обоснование
SQLite упрощает локальную разработку без необходимости PostgreSQL.

## 2. Сценарий
Разработчик запускает приложение локально с SQLite.

## 3. Требования
- Адаптер с тем же интерфейсом, что и PostgreSQL.
- Автоматическое создание схемы.
- Переключение через конфигурацию.

## 4. Критерии приёмки
- [x] SQLite адаптер реализован.
- [x] Интерфейс совместим с PostgreSQL адаптером.
- [x] Автоматическая миграция при старте.
- [x] Тесты проходят на SQLite.
